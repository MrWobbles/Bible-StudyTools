# Images Folder

Store class-related images here:
- Lesson graphics
- Diagrams and charts
- Photo resources
- Icons and illustrations

These files will be tracked in git and can be referenced in `classes.json`.

## Example References

In `classes.json`, you can reference images like:
```json
{
  "resources": [
    {
      "type": "image",
      "title": "Timeline Chart",
      "path": "assets/images/class1-timeline.png",
      "description": "Biblical manuscript timeline"
    }
  ]
}
```
