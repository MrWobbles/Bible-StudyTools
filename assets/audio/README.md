# Audio Folder

Store audio files here (NOT tracked in git due to size).

## Usage

- Audio recordings of lessons
- Supplemental audio content
- Music or sound effects

## Example References

In `classes.json`, reference audio files:
```json
{
  "media": [
    {
      "type": "audio",
      "title": "Audio Recording",
      "path": "assets/audio/class1-recording.mp3",
      "note": "Audio-only version of the lesson"
    }
  ]
}
```
