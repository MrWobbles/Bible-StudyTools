# Documents Folder

Store class-related documents here:
- PDFs (handouts, reference materials, reading guides)
- Word documents
- Text files
- Any supplemental reading materials

These files will be tracked in git and referenced in `classes.json`.

## Example References

In `classes.json`, you can reference documents like:
```json
{
  "resources": [
    {
      "type": "pdf",
      "title": "Study Guide",
      "path": "assets/documents/class1-study-guide.pdf"
    }
  ]
}
```
